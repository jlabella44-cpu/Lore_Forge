doesn't re-learn them:

  branch; `git fetch` + rebase before every push or the push
  rejects. Hit twice in this session (the `a47848b` parallel commit
  rebased through cleanly both times).

  foreground ones; serialise when commands `cd`. Hit the "Parallel
  bash tool runs share cwd" warning from 04-19 still applies.

  `jlabella44-cpu/lore_forge.git`; GitHub redirects to
  `Lore_Forge.git`. Push works but emits "This repository moved"
  every time. Low priority to rewrite origin.
